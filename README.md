Edu Cart is an online platform designed for students who want fast access to educational products and learning materials. The system presents books, courses, and study resources in a clean digital store. Students browse products, view details, place orders, and track purchases through a simple interface. The platform focuses on ease of use, organized product categories, and a smooth checkout process.

Key features of Edu Cart.

• Student friendly online store for educational resources 📚
• Product catalog with categories and search options 🔎
• Secure user signup and login system 🔐
• Shopping cart and order management system 🛒
• Admin dashboard for managing products, orders, and users ⚙️
• Order status tracking with timestamps for order progress ⏱️
• Clean interface designed for quick navigation and usability

Purpose of the project.
Edu Cart demonstrates how an online shopping system works in a real environment. The project shows database integration, user authentication, product management, and order processing within a modern web application structure. 🎯
